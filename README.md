# TechMeme :boom:

## About

本项目旨在个人写技术文章时快速使用有趣的技术表情包，避免搜索心仪的表情包而使思路中断。

meme (/ˈmiːm/), 是指在同一个文化氛围中，人与人之间传播的思想、行为或者风格。
接地气的翻译就是梗、表情包。

> [wikipedia](https://en.wikipedia.org/wiki/Internet_meme)
